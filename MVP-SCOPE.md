# Escopo do MVP

## Must Have (Obrigatório)
* Landing Page com seção Hero focada em "Engenharia de Software".
* Seção "Institucional/Saúde" com descrição de expertise governamental.
* Vitrine dinâmica de produtos (cards com imagem, título, tech stack e preço/valor).
* Página de detalhes simples para cada produto/projeto.
* Botão flutuante de WhatsApp.

## Should Have (Desejável)
* Dark/Light mode toggle (foco inicial no Light).
* Seção de "Skills" com ícones modernos (shadcn/ui badges).

## Won't Have (Futuro)
* Checkout de pagamento interno.
* Área de login para clientes.
* Dashboard administrativo complexo (uso do painel do Supabase diretamente).

## Hipóteses a Validar
1.  A autoridade gerada pela seção "Gov" aumenta a taxa de conversão de clientes privados.
2.  A vitrine de produtos gera leads passivos sem canibalizar a consultoria de alto valor.
